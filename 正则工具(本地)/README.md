# 正则表达式匹配与高亮工具

一个简洁、高效、实时的在线正则表达式测试工具，支持源数据文本匹配、高亮显示以及结果导出。

## 功能特点

- **实时匹配**：输入正则表达式即可实时看到匹配结果。
- **文本高亮**：源文本中的匹配项采用橙色高亮显示，位置精准。
- **结果统计**：自动统计匹配到的总次数。
- **一键复制**：支持将所有匹配到的结果一键复制到剪贴板。
- **简洁 UI**：清爽的界面设计，支持响应式布局，不拥挤。

## 技术实现

- **HTML5**: 构建页面结构。
- **CSS3**: 负责界面美化与高亮对齐逻辑。
- **JavaScript**: 处理正则表达式匹配核心逻辑。

## 使用方法

1. 直接在浏览器中打开 `index.html`。
2. 在“正则表达式”输入框中输入您的正则模式。
3. 在“源数据文本”区域输入或粘贴需要匹配的文本。
4. 在右侧（或下方）查看匹配结果列表，并根据需要点击“复制结果”。

## 文件结构

- `index.html`: 网页主文件。
- `styles.css`: 样式表，包含高亮同步的核心 CSS。
- `script.js`: 业务逻辑，包含正则匹配与 DOM 更新。
- `requirements.md`: 项目需求文档。
- `.gitignore`: Git 忽略配置文件。
