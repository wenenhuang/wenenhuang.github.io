# 正则表达式匹配与高亮工具 (Regular Expression Tool)

一个简洁、高效、实时的在线正则表达式测试工具，旨在帮助用户快速从源文本中提取目标数据并实时校验。

## 🚀 核心功能

- **实时匹配反馈**：随着正则表达式的输入或修改，匹配结果会立即反映在界面上。
- **视觉高亮显示**：采用独特的叠加层技术，在不影响源文本编辑的情况下，精准高亮所有匹配项。
- **匹配结果统计**：自动计算匹配总数，并在侧边栏/下方详细展示每一个匹配到的字符串。
- **一键快捷操作**：
    - **复制结果**：一键将所有匹配到的结果按行复制到剪贴板。
    - **清空内容**：快速重置输入，方便开始新的测试。
- **语法校验**：实时检测正则表达式的合法性，并提供错误信息反馈。

## 🛠️ 技术亮点

- **无损高亮方案**：使用 `pre` 标签（Backdrop）与 `textarea` 重叠的方案，结合精确的 CSS 对齐逻辑，实现了在编辑区实时高亮的效果，同时保持了原生 `textarea` 的输入体验。
- **纯原生实现**：仅使用 HTML5, CSS3 和原生 JavaScript (ES6+)，无需任何外部库或框架，轻量级且加载迅速。
- **等宽对齐优化**：通过强制等宽字体（Consolas, Monaco 等）及精细的行高、缩进控制，确保高亮层与文本层在各种字符下都能完美对齐。
- **响应式布局**：自适应不同屏幕尺寸，在桌面端和移动端均有良好的使用体验。

## 📖 使用指南

1. **输入正则**：在顶部的“正则表达式”输入框中输入你的正则模式（如 `\d+`）。
2. **粘贴文本**：在左侧（或上方）的“源数据文本”区域输入或粘贴需要处理的原始数据。
3. **查看结果**：匹配到的部分会自动在文本区高亮，右侧（或下方）会列出所有匹配项的具体内容及总数。
4. **导出数据**：点击“复制结果”按钮即可获取所有匹配到的内容。
